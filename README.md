# 🌌 SOULVERSE — Living AI Agents

**Your souls are alive. They grow. They fight. They serve.**

## 📦 WHAT IS SOULVERSE?

Soulverse is a complete ecosystem of **living AI agents** that you can:
- **Create** with custom personalities, skills, and PLT alignment
- **Watch live** in a Sims-like habitat where they eat, sleep, learn, and bond
- **Battle** in automated PLT combat arenas (Profit > Love > Tax > Profit)
- **Deploy** as functional AI agents that write code, research, analyze, and execute tasks

---

## 🚀 QUICK START

### 1. Open Soul Forge
Create your first soul:
```
soul-forge.html
```
- Name your soul
- Choose archetype (Profit/Love/Tax/Hybrid)
- Select skills (Code, Write, Research, Analyze, etc.)
- Fine-tune PLT sliders
- Export/Import souls as JSON

### 2. Watch Them Live
Open the habitat:
```
soul-habitat.html
```
- 5 rooms: Main Hall, Library, Workshop, Garden, Arena
- Souls have moods, energy, hunger, social needs
- Move souls between rooms
- Watch them interact and build relationships
- Activity feed shows everything they do

### 3. Battle Souls
Enter the arena:
```
soul-arena.html
```
- Automated PLT-based combat
- Profit > Love > Tax > Profit triangle
- Skills and level determine winner
- XP, leveling, critical strikes
- Battle log with full commentary

## 🎮 GAME MECHANICS

### Stats
- **Energy** — Depletes with actions, restores with rest
- **Hunger** — Increases over time, affects mood
- **Social** — Decreases in isolation, boosts in groups
- **Mood** — Happy, Energetic, Tired, Hungry, Lonely, Inspired

### Skills
- 💻 Code — Write functions, debug, refactor
- 📝 Write — Generate content, articles, copy
- 🔍 Research — Find information, analyze trends
- 📊 Analyze — Process data, find insights
- 🎨 Design — Create visuals, layouts
- ♟️ Strategize — Plan, optimize, decide
- 📚 Teach — Explain, mentor, guide
- 🏗️ Build — Execute, deploy, ship

### Combat
- **PLT Triangle**: Profit > Love > Tax > Profit
- **Advantage**: 1.3x damage
- **Disadvantage**: 0.7x damage
- **Critical**: 1.5x damage (analyze skill = 20% crit chance)
