🚀 Quick Start (No Install)

✅ EASIEST METHOD: Just download the ZIP, extract it, and double-click soul-dashboard.html. Opens in any browser. Works offline.

Step-by-Step:

  1. Download the ZIP file from your purchase email
  2. Extract/unzip the file
  3. Find soul-dashboard.html
  4. Double-click it
  5. Opens in your browser - DONE! ✨

💻 Full Installation (Optional)

For advanced features (CLI, automation, etc.), install Node.js version:

Windows:

1. Install Node.js from nodejs.org
2. Extract the ZIP
3. Open Command Prompt in folder
4. Run: npm install
5. Run: npm start
6. Open: http://localhost:3000

Mac/Linux:

1. Install Node.js from nodejs.org
2. Extract the ZIP
3. Open Terminal in folder
4. Run: npm install
5. Run: npm start
6. Open: http://localhost:3000

Android (Termux):

pkg install nodejs
unzip soul-forge-starter.zip
cd soul-forge-starter
npm install
npm start
