## 🚀 INSTALLATION (By Platform)

### **Windows**

#### Option 1: Browser (Easiest - No Install)
1. Download ZIP file
2. Extract to any folder
3. Double-click `soul-dashboard.html`
4. Opens in your browser - DONE!

#### Option 2: Full App (Node.js)
1. Install Node.js: https://nodejs.org
2. Download ZIP
3. Extract
4. Open Command Prompt in folder
5. Run: `npm install`
6. Run: `npm start`
7. Opens at: http://localhost:3000

## 🆘 TROUBLESHOOTING

### **"File won't open"**
- Make sure you extracted the ZIP first
- Right-click → Extract All (Windows)
- Double-click (Mac)

### **"Dashboard won't load"**
- Try a different browser
- Clear browser cache
- Make sure file path has no special characters

### **"Node.js command not found"**
- Install Node.js from nodejs.org
- Restart terminal/command prompt
- Check installation: `node --version`

### **"npm install fails"**
- Delete `node_modules` folder
- Delete `package-lock.json`
- Run `npm install` again
